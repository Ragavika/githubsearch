Dev-Detective: GitHub User Search App

"Dev-Detective" is a clean, responsive web application that allows users to search for any GitHub developer and instantly retrieve their profile details and latest repositories. 

This project was built to master the integration of external APIs using JavaScript, focusing on asynchronous programming and DOM manipulation.

Live Demo
https://githubsearch-nu.vercel.app/

Features
- Real-time Search: Look up any GitHub user by their username.
- Dynamic Profile Cards: Displays Avatar, Name, Bio, Join Date, and Portfolio URL.
- Repository Integration: Fetches and displays the top 5 latest public repositories.
- Robust Error Handling: Provides user-friendly feedback if a user is not found.
- Responsive Design: Optimized for mobile and desktop screens.

Tech Stack
- HTML5: Semantic structure.
- CSS3: Modern layouts using Flexbox and Grid.
- JavaScript (ES6+): - `fetch()` API for data retrieval.
    - `Async/Await` for clean asynchronous code.
    - DOM Manipulation to dynamically update the UI.
